开发者文档

开发之前，你需要有 ES6、SVG的相关知识！

本库提供了许多基础设施，
Base类类提供了许多基于es6的实用方法，本库的底层基于Base类。
SVG类提供了许多绘图的实用方法，chartbase类封装了更高层的绘制方法，用于方便的绘制图表中的图形
